# CopyFiles

Devido ŕ dificuldade em utilizar ferramentas como Google Drive para backup de arquivos em um dispositivo Android (seja por limitaçőes técnicas ou conhecimento), este projeto foi desenvolvido para simplificar o processo.

Seu objetivo é fornecer um código simples em C# que automatiza a transferęncia de arquivos de um celular Android para um SSD, facilitando o backup local sem depender de soluçőes em nuvem.
